## 段落（Paragraphs）

これは通常の段落です。複数の文章が含まれています。
改行は自動的に結合されて一つの段落になります。

段落を分けるには、空行を挟みます。

行末に2つのスペースを入れることで  
改行することもできます。

## リスト（Lists）

### 順序なしリスト（Unordered List）

- 項目1
- 項目2
  - サブ項目2.1
  - サブ項目2.2
    - サブサブ項目2.2.1
- 項目3

### 順序ありリスト（Ordered List）

1. 最初の項目
2. 二番目の項目
   1. サブ項目2.1
   2. サブ項目2.2
      1. サブサブ項目2.2.1
3. 三番目の項目

### タスクリスト（Task List）

- [x] 完了したタスク
- [ ] 未完了のタスク
- [x] もう一つの完了したタスク
- [ ] 別の未完了のタスク

## コードブロック（Code Blocks）

### インラインコード

変数 `const example = 'hello world';` の例です。

### フェンスコードブロック

```javascript title="greet.js"
// JavaScript の例
function greet(name) {
	console.log(`Hello, ${name}!`);
}

greet("World");
```

```python title="fibonacci.py"
# Python の例
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```

```bash title="Shellの例"
echo "Hello, World!"
ls -la
```

これは脚注のサンプルです[^1]。複数の脚注も可能です[^note2]。
